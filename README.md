# Altaf GBIAN 001 - Professional Football Portfolio

This is an extreme, professional portfolio website for the elite footballer Altaf GBIAN 001. Built with modern web technologies for maximum impact and user engagement.

## Features

- **Responsive Design**: Optimized for all devices
- **Smooth Animations**: Engaging transitions and effects
- **Interactive Elements**: Hover effects, lightbox gallery, animated counters
- **Modern UI**: Clean, professional design with gradient backgrounds
- **SEO Optimized**: Proper meta tags and structure
- **Fast Loading**: Optimized for performance

## Sections

1. **Hero**: Eye-catching introduction with call-to-action and particle background
2. **About**: Personal story and key achievements
3. **Stats**: Career statistics with animated counters and 3D tilt effects
4. **Achievements**: Trophy case with awards and accomplishments
5. **Gallery**: Photo gallery with lightbox and hover effects
6. **Testimonials**: Quotes from coaches, teammates, and analysts
7. **Contact**: Contact form and social links

## Setup Instructions

1. **Images**: The portfolio currently uses placeholder images from Unsplash. Replace the `src` attributes in `index.html` with your own image URLs or local files.
   - Hero image: Update the `img` in `.hero-image`
   - About portrait: Update the `img` in `.about-image`
   - Gallery: Update the 6 images in `.gallery-grid`

2. **Customize Content**:
   - Edit `index.html` to update personal information, stats, achievements, and testimonials
   - Modify `styles.css` for color scheme or styling changes
   - Update `script.js` if needed for additional functionality

3. **Deploy**:
   - Upload all files to a web server
   - For local development, open `index.html` in a browser
   - Recommended: Use GitHub Pages, Netlify, or Vercel for hosting
